# Changelog

## 0.1.0

- Rename the extension to `pionus.vim-terminal-editor`.
- Add `pionus.vimTerminalEditor.editor` for choosing `vim` or `nvim`.
- Add the editor-neutral `pionus.vimTerminalEditor.openCurrentFile` command.
- Retain `pionus.nvimTerminalEditor.openCurrentFile` as a compatibility alias.

## 0.0.2

- Make the extension discoverable by searching for Pionus in VS Code.

## 0.0.1

- Initial public release with Neovim terminal editing, diagnostic navigation,
  and rendered-preview commands.
